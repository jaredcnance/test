Examples

These requests have been generated against the "GettingStarted" application and are updated on every deployment.

All of these requests have been created using out-of-the-box features.

Simple CRUD

Create

curl -vs http://localhost:5001/api/people           \
    -H "Accept: application/vnd.api+json"           \
    -H "Content-Type: application/vnd.api+json"     \
    -d '{
            "data": {
                "type": "people",
                "attributes": {
                    "name": "Alice"
                }
            }
        }'

HTTP/1.1 201 Created

{
  "data": {
    "attributes": {
      "name": "Alice"
    },
    "relationships": {
      "articles": {
        "links": {
          "self": "http://localhost:5001/api/people/1/relationships/articles",
          "related": "http://localhost:5001/api/people/1/articles"
        }
      }
    },
    "type": "people",
    "id": "1"
  }
}

Create with Relationship
